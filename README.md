# WordPress Themes: Estate Pro & Tech Review

Цей репозиторій містить дві кастомні WordPress теми: **Estate Pro** та **Tech Review**. Головний акцент у розробці цих тем зроблено на використанні сучасних підходів до створення інтерфейсів за допомогою JavaScript, кастомних Gutenberg блоків та потужних можливостей плагіна Advanced Custom Fields (ACF).

---

## 🏢 Estate Pro
**Estate Pro** — це спеціалізована тема для агентств нерухомості, ріелторів та каталогів власності. Вона забезпечує зручний пошук, фільтрацію та детальне відображення об'єктів нерухомості.

## 💻 Tech Review
**Tech Review** — це тема, орієнтована на технологічні блоги, сайти з оглядами гаджетів та IT-новин. Вона оптимізована для читабельності, швидкого завантаження та інтерактивної взаємодії з контентом.

---

## 🚀 Ключові технології та підходи

### 1. JavaScript (Ванільний JS та ES6+)
У розробці обох тем активно використовується JavaScript для забезпечення динамічного та інтерактивного користувацького досвіду:
- **Асинхронні запити (AJAX/Fetch):** Динамічне завантаження контенту (наприклад, фільтрація нерухомості в Estate Pro без перезавантаження сторінки).
- **Інтерактивні UI компоненти:** Слайдери, модальні вікна, кастомні селекти та інтерактивні карти.
- **Оптимізація продуктивності:** Використання сучасного синтаксису ES6+, мінімізація та модульна структура коду для швидкого завантаження скриптів.

### 2. Кастомні елементи та Gutenberg блоки
Замість використання важких page-builder'ів, розробка базується на нативному редакторі WordPress — **Gutenberg**:
- **Кастомні блоки:** Створено унікальні блоки під специфічні потреби кожної теми (наприклад, блок "Картка об'єкту нерухомості" для Estate Pro або "Технічні характеристики" для Tech Review).
- **Реактивність у редакторі:** Блоки максимально наближені до того, як вони виглядають на фронтенді, що забезпечує зручний досвід редагування (WYSIWYG).
- **Секції:** Розроблені готові патерни та секції для швидкого збирання сторінок з кастомних елементів.

### 3. Advanced Custom Fields (ACF) Плагін
Плагін **ACF Pro** є ядром для управління даними в обох темах:
- **Інтеграція з Gutenberg (ACF Blocks):** Всі кастомні блоки розроблені за допомогою ACF Blocks, що дозволяє легко створювати складні структури даних без глибокого занурення в React.
- **Гнучке налаштування:** Використання Options Pages для глобальних налаштувань тем (контакти, посилання на соцмережі, логотипи тощо).

### 4. Поля ACF (ACF Fields)
Для максимальної гнучкості контенту використовуються різноманітні типи полів ACF:
- **Repeater Fields (Повторювачі):** Для створення списків переваг, галерей зображень об'єктів нерухомості, або списків плюсів/мінусів в оглядах (Tech Review).
- **Flexible Content (Гнучкий контент):** Дозволяє адміністраторам створювати унікальні лейаути сторінок, комбінуючи різні типи контенту.
- **Relational Fields (Відносини):** Зв'язування постів між собою (наприклад, прив'язка агента до конкретного об'єкту нерухомості).
- **Custom Field Groups:** Чітко структуровані групи полів, прив'язані до конкретних Custom Post Types (CPT) та шаблонів сторінок.

---

## 🛠 Встановлення та розгортання
1. Клонуйте або завантажте цей репозиторій у папку `wp-content/themes/` вашого WordPress сайту.
2. Переконайтеся, що плагін **Advanced Custom Fields PRO** встановлено та активовано.
3. Активуйте потрібну тему (Estate Pro або Tech Review) в панелі адміністратора WordPress (`Вигляд` -> `Теми`).
4. (Опціонально) Імпортуйте налаштування ACF полів з папки `acf-json` для кожної теми, якщо вони не підтягнулися автоматично.
